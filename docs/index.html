CODE EXAMPLE

Classic BASIC syntax that feels just like the old days

pong.bas 
' Simple Pong Game in PanicBasic
SCREEN 13
CLS

CONST PADDLE_HEIGHT = 40
CONST BALL_SIZE = 4

ballX = 160
ballY = 100

DO
    CLS

    ' Draw paddle
    LINE (10, paddleY)-(15, paddleY + PADDLE_HEIGHT), 15, BF

    ' Draw ball
    CIRCLE (ballX, ballY), BALL_SIZE, 14
    PAINT (ballX, ballY), 14

    ' Move ball
    ballX = ballX + ballDX
    ballY = ballY + ballDY

    _DELAY 0.016
LOOP

QUICK START

Get up and running in minutes

1

Clone Repository

Download the source code from GitHub

git clone https://github.com/zerojnt/pasic.git
2

Install Dependencies

Install the required Python packages

pip install -r requirements.txt
3

Launch Editor

Start the integrated environment

python main.py
4

Run a Game

Try one of the included examples

python main.py --run examples/pong.bas

COMMANDS

Supported BASIC commands and functions

Graphics

  • SCREEN Set graphics mode
  • CLS Clear screen
  • LINE Draw lines/boxes
  • CIRCLE Draw circles
  • PAINT Flood fill
  • PSET Set pixel

Flow Control

  • IF...THEN Conditionals
  • FOR...NEXT For loops
  • DO...LOOP Do loops
  • WHILE...WEND While loops
  • SELECT CASE Switch
  • GOSUB Subroutines

Variables

  • DIM Declare arrays
  • CONST Define constants
  • LET Assign values

I/O & Timing

  • PRINT Output text
  • INPUT User input
  • INKEY$ Get key press
  • _DELAY Non-blocking wait
  • SLEEP Blocking wait
