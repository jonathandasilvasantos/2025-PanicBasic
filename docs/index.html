PanicBasic

A retro-style BASIC programming environment in Python

View on GitHub Download ZIP

Features

📝

Integrated Editor

Custom text editor with line numbering, undo/redo, cut/copy/paste, and a unique drawing layer for annotations.

💻

BASIC Interpreter

Supports classic commands like SCREEN, CLS, PRINT, LINE, CIRCLE, PAINT, loops, conditionals, and subroutines.

🎮

Retro Graphics

320x200 software surface emulating classic BASIC graphics modes with 256 colors.

🖨

Built-in Console

Shell-like console for running system commands directly within the application.

🎳

Example Games

Includes Pong, Catch the Star, Car Racing, Pinball, and Endless Runner games.

🚀

Easy to Use

Simple installation with pip, intuitive interface, and comprehensive documentation.

Code Example

' Simple Pong Game in PanicBasic
SCREEN 13
CLS

CONST PADDLE_HEIGHT = 40
CONST BALL_SIZE = 4

ballX = 160
ballY = 100

DO
    CLS

    ' Draw paddle
    LINE (0, paddleY)-(10, paddleY + PADDLE_HEIGHT), 15, BF

    ' Draw ball
    CIRCLE (ballX, ballY), BALL_SIZE, 15
    PAINT (ballX, ballY), 15

    ' Move ball
    ballX = ballX + ballDX
    ballY = ballY + ballDY

    _DELAY 0.016
LOOP

Quick Start

1

Clone the Repository

Download the source code from GitHub

git clone https://github.com/zerojnt/pasic.git
2

Install Dependencies

Install required Python packages

pip install -r requirements.txt
3

Run PanicBasic

Launch the integrated environment

python main.py
4

Run an Example

Try one of the included games

python main.py --run examples/pong.bas

Supported Commands

Graphics

  • SCREEN Set graphics mode
  • CLS Clear screen
  • LINE Draw lines/boxes
  • CIRCLE Draw circles
  • PAINT Flood fill
  • PSET Set pixel

Flow Control

  • IF...THEN Conditionals
  • FOR...NEXT For loops
  • DO...LOOP Do loops
  • WHILE...WEND While loops
  • SELECT CASE Switch
  • GOSUB Subroutines

Variables

  • DIM Declare arrays
  • CONST Define constants
  • LET Assign values

I/O & Timing

  • PRINT Output text
  • INPUT Get user input
  • INKEY$ Get key press
  • _DELAY Non-blocking delay
  • SLEEP Blocking delay
